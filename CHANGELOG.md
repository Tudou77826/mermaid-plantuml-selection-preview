# Changelog

## 0.7.0 - 2026-07-16

- 点击扩展图标时自动读取当前页面及可访问 iframe 的选区并预览。
- 无法读取标准选区时显示极简粘贴框，支持 `Ctrl+Enter` 提交。
- 增加 `Ctrl+Shift+M` 快捷入口，并允许用户在 Chrome 中自定义。
- 保留右键菜单和受保护页面的独立预览回退。
- 不增加全站访问、剪贴板读取或常驻内容脚本权限。

## 0.6.0 - 2026-07-14

- 支持 Mermaid 与 PlantUML 图表自动识别和本地渲染。
- 支持带代码围栏和不带围栏的选区。
- 提供留在原页面上下文中的预览浮层。
- 支持缩放、拖动、适应窗口和键盘快捷键。
- 为受保护页面提供独立预览窗口回退。
- 增加最小权限配置、一次性选区清理和 PlantUML 外部内容限制。
- 增加单元测试、浏览器端到端测试、图标生成和 Release 打包流程。
